# Academic Paper Revision (Five Methods)

## When to Use
- Revising a completed draft manuscript
- Incorporating peer review or editorial feedback
- Improving paper quality before resubmission
- Systematic editing of academic manuscripts

## When NOT to use

- 不适用于论文初稿撰写或学术写作零基础场景（需已有完整草稿）
- 当用户没有提供足够信息来判断适用性时，先询问而非假设
- 如果已有更专门的 skill 覆盖当前场景，优先使用那个 skill
- 不要在时间极度紧迫时跳过 skill 的核心流程——宁可缩短范围也不要跳过必要步骤

## Prerequisites
- Completed draft manuscript
- Review comments (if available)
- Understanding of journal requirements

## The Five Methods (五字法)

### 1. 调 (Tiáo) - Adjust Theme and Structure

**Theme Adjustment:**
- Verify theme is prominent and clear
- Confirm viewpoints align with objective reality
- Check for one-sided or absolutist analysis
- Ensure main purpose is well-expressed

**Structure Adjustment:**
- Audit layout for clarity and logical flow
- Address uneven weight distribution
- Fix unclear organization (脉络)
- Resolve tangled logic
- Make structure balanced and refined

### 2. 换 (Huàn) - Change Methods and Replace Materials

**Writing Style:**
- Break old patterns and routines
- Transform writing angles and methods
- Create freshness in presentation

**Material Audit:**
- Check alignment with theme and viewpoints
- Verify materials are typical and representative
- Ensure effective support for arguments
- Replace unsuitable materials
- Achieve theory-evidence integration

### 3. 改 (Gǎi) - Correct Errors and Improve Style

**Language Precision:**
- Correct inappropriate or imprecise expressions
- Fix wrong sentences, words, typos
- Correct punctuation errors
- Verify data accuracy

**Style Improvement:**
- Strive for "short, real, new" (短实新)
- Eliminate word games and ornamental language
- Remove forced parallelism
- Delete unnecessary embellishment
- Use accessible, plain, natural expression

### 4. 增 (Zēng) - Add Content and Supplement Elements

**Content Completeness:**
- Ensure closed loop: raise problem → analyze problem → solve problem
- Check for missing or incomplete sections
- Fill gaps in argumentation

**Element Verification:**
- Confirm all required academic elements present
- Supplement missing essential components
- Add supporting data or evidence where needed

### 5. 删 (Shān) - Delete Layers and Remove Redundancy

**Layer Simplification:**
- Avoid excessive hierarchy
- Reduce complexity that confuses readers
- Follow principle: "refine where possible, simplify where possible"

**Content Pruning:**
- Delete repetitive content without hesitation
- Remove unnecessary modifiers
- Cut empty phrases and clichés
- Eliminate redundant expressions

## Assessment Variables

| Variable | Type | Description |
|----------|------|-------------|
| theme_clarity | assessment | Degree of theme prominence |
| structure_balance | assessment | Whether structure is well-balanced |
| material_relevance | assessment | Alignment with theme/arguments |
| language_precision | assessment | Accuracy of expressions |
| content_completeness | assessment | Presence of all required elements |

## Output
A systematically revised academic paper with improved clarity, structure, materials, language, completeness, and conciseness.

## Gotchas

- **不要跳过"调"直接"改"**：五字法必须从"调主题→调结构"开始，上来就改句子会白改——结构变了句子要重写
- **不要把"删"变成"缩"**：删减不是把长句变短句，是彻底去掉没有论证作用的段落——犹豫就先标记，最后统一删
- **不要在"增"的时候偏离主题**：补充材料最容易犯的错是"加更多证据"而非"补缺的逻辑环节"
- **不要把"换"当成重写**：换材料不是推翻重来，是找到能更精准支撑论点的现有材料或数据
- **不要忽略"短实新"原则的优先级**：风格改法优先级是：删废话 > 换说法 > 调结构，不要反过来

## Risk Blacklist

- **修改论文时未备份**：五字法涉及大规模结构调整，没有版本备份等于裸奔
- **接受"改后比原来长"**：好论文改完应该更短，变长说明你在加废话
- **在审稿意见还没读透就动手改**：每条审稿意见必须分类（必须改 / 可以不改 / 不需要改）后再动笔
